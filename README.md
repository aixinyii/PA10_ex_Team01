# PA10_ex_Team01

PA拾焰队的中国城市财政赤字分析

## github 使用简介
### 注册账号
自己 sign up 一下就好。

### 如何加入项目
项目建立人在该仓库页面上面点 setting，然后在左边点 collaborators，然后点 Add people，最后填写被邀请人的id，发送邀请即可。如下图所示：
![alt text](image.png)
### 如何在自己的电脑上编辑项目
#### 加入项目
找你的项目建立人把你加进 collaborators。
#### 下载安装紫猫
- 下载 Github Desktop, 官方下载链接：https://desktop.github.com/download/
- 下载完了就打开安装包，安装即可。
- 默认安装的话，应该在文档里面的 Github 文件夹里。如下图所示：
![alt text](image-1.png)
#### 登录紫猫
- 打开紫猫，登录你的 Github 账号
#### folk 主仓库
- 进入主项目主页（比如本项目为：https://github.com/enderkidleaf/PA10_ex_Team01）
- 点folk，如图所示：![alt text](image-2.png)
#### 在紫猫里克隆项目
- 在紫猫里点左上角的 file，然后点 clone repository，如图所示：![alt text](image-3.png)
- 选择 folk 来的项目，然后选择目录，如图所示：![alt text](image-4.png)
- 等下载完成
